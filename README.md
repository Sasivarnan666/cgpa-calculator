# 🎓 CGPA Calculator (CSE)

A Python-based GPA & CGPA calculator for CSE students.

## 🚀 Features
- Calculate Semester GPA
- Calculate CGPA
- Menu-driven CLI system
- Modular code structure

## 🛠 Tech Stack
- Python
- Git
- VS Code

## 📂 Project Structure
- main.py → control flow
- gpa.py → GPA logic
- cgpa.py → CGPA logic
- data.py → subjects & credits
- utils.py → helper functions

## ▶️ How to Run

```bash
python main.py
